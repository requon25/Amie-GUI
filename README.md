Amie-GUI es una interfaz gráfica diseñada para facilitar la interacción con modelos de lenguaje locales. 
El proyecto integra la potencia de la Inteligencia Artificial con el framework PySide6, ofreciendo una experiencia de usuario fluida y visual.


Requisitos e Instalación:
pip install ollama langchain PySide6


Uso del Proyecto:

1. Selección del modelo: El usuario puede elegir y descargar cualquier modelo compatible desde la página oficial de Ollama.
2. Ejecución: Una vez configurado el modelo, simplemente inicie la aplicación ejecutando el archivo main.py
